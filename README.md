## Installation Guide

### Prerequisites
1. **Node.js**: Ensure you have Node.js installed on your machine. You can download it [here](https://nodejs.org/).
2. **Expo CLI**: Install Expo CLI globally on your machine using npm:

```bash
npm install -g expo-cli
```

### Steps
1. **Clone the Repository**: Open your terminal and navigate to the directory where you want to download the project. Then, run the following command to clone the repository:
```bash   
git clone https://github.com/manasa8910/musify.git
```


2. **Navigate to Project Directory**: Change your current directory to the newly cloned `musify` directory by running:
```bash   
cd musify
```

3. **Install Dependencies**: Use npm (Node Package Manager) to install the project dependencies. Run the following command:
```bash   
npm install
```

This will install all the necessary packages and dependencies required for the project to run.

4. **Start the Application**: Once the dependencies are installed, you can start the application. Use the following command:
```bash   
npm start
```
This will start the Expo development server and open a new browser tab with the Expo DevTools.

5. **Running on Emulator/Simulator**: If you want to run the app on an emulator or simulator, make sure it's set up and running. You can start the emulator/simulator from Android Studio (for Android) or Xcode (for iOS).

6. **Running on Physical Device**: To run the app on a physical device, you need to install the Expo Go app from the App Store (for iOS) or Google Play Store (for Android). Then, scan the QR code displayed in the Expo DevTools browser tab using the Expo Go app.

7. **Explore Musify**: You're all set! Explore the features and functionalities of Musify on your emulator, simulator, or physical device.
